KEY ARCHITECTURE DECISIONS
═══════════════════════════════════════════════════════════════

- Dashboard = RESEARCH CENTER (scan, filter, identify)
- Chart Wall = ACTION CENTER (watch, wait for entry)
- Modal = CONFIRM (scorecard, criteria, execute decision)
- TradeStation = EXECUTE (pull trigger)

- Universal Card: one uCard() function, all sections call it
- All sections compact mode, click opens modal
- Setup scoring: independent evaluation, no waterfall bias
- Instant modal: client-side data, server refresh on demand
- Chart Wall: Lightweight Charts (open source, full indicator control)
- Quota/API charts: 5-day scrollable history

LESSONS LEARNED
═══════════════════════════════════════════════════════════════
