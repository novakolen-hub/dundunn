====================================================
KEY DECISIONS (Mar 10 session)
====================================================

1. SIZING: % of equity, not fixed dollars
2. DEPLOYED CAPITAL: Hybrid — size per total equity, cap total deployment
3. BE DISCOUNT: Swings at breakeven count at 50% against cap
4. LOSS LIMITS: Advisory not hard block
5. GP TRANSITIONS: Grandfather existing, only block new entries
6. PAPER TRADES: Exempt from deployment caps
7. PORTFOLIO VIEW: Theme-based clusters as default
8. DT→SW: Formal conversion with full carryover
9. NEWS: 30-min cache, 3 feeds only, no per-ticker feeds
10. TIMING: Dynamic DST detection, never hardcode AZ hours
11. ARETE: Reference only — alignment check, not system override
12. DASHBOARD: Top-down flow (Market → Sectors → Stocks → Intel → Positions)
13. YOUTUBE: Inline desktop, app on mobile
